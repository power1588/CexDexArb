# 套利监控 MVP Tasks

## 文档说明

- 本文档基于以下资料整理：
  - `套利监控MVP-产品需求文档.md`
  - `套利监控MVP-技术架构文档.md`
  - `套利监控MVP-TDD开发待办清单.md`
- 当前范围仅覆盖 HTML 单文件 MVP。
- 所有执行类交互均为演示行为，不接真实 API，不执行真实交易。
- 开发方式遵循 TDD：先写失败测试，再实现最小代码，再重构。
- 当前勾选状态已按项目当前代码实现与验证结果更新；未勾选项表示尚未实现、未完全闭环，或尚未完成正式验证。
